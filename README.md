# Boston House Price Prediction

## Project Description
This project implements machine learning models to predict Boston house prices using various features.

## Features
- Data exploration and visualization
- Data preprocessing and feature scaling
- Multiple regression models (Linear Regression, Random Forest)
- Model evaluation using R², RMSE, MAE
- Feature importance analysis
- Visualizations for better insights

## Dataset
The Boston Housing dataset contains information about:
- CRIM: Crime rate
- RM: Average number of rooms
- LSTAT: Lower status population
- And other housing features

## Results
- Best Model: Random Forest
- R² Score: ~0.88
- RMSE: ~4.5

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run the script: `python boston_price_prediction.py`

## Requirements
- Python 3.8+
- Libraries listed in requirements.txt
